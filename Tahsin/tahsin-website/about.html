About Tahsin

Tahsin is a responsive service website designed to connect users with suitable Quran learning classes and teachers. Its main idea is simple: users first choose a class route, then receive a teacher list that can be searched, filtered, sorted, and intelligently ranked.

What makes it unique? Tahsin does not stop at listing teachers. It includes a smart match score so the system recommends the most suitable teacher based on user preference.

Technology used

  • HTML: semantic structure with accessible navigation and forms.
  • CSS: consistent dark green, white, and gold theme with responsive layouts.
  • JavaScript: search, sorting, filtering, matching logic, and compare feature.
  • AJAX: fetch() is used to retrieve JSON data without reloading pages.
  • JSON: class, teacher, and team data are stored in JSON files.
  • API: Leaflet and OpenStreetMap display teacher locations on a map.

Main objectives

  • Help users discover suitable Quran learning services
  • Provide class-based navigation before teacher selection
  • Support searching, sorting, and filtering clearly
  • Show dynamic content with JSON and AJAX
  • Demonstrate an API page with practical value
  • Maintain accessibility and responsive design
