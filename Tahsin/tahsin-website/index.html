Why Tahsin stands out

This is not just a listing website. It combines guided class selection with a smart ranking system to recommend teachers more meaningfully.

🧭

Guided Class Discovery

Users start with the class they actually want such as Iqra, Hifz, Tarannum, Qiraat, OKU support, or advanced talaqqi tracks.

🤖

Smart Match Score

The teacher page ranks options using class fit, budget, mode, level, availability, rating, experience, and support needs.

🗺️

Interactive Map View

A separate API page maps teacher locations to make discovery visual and more practical for users.

Quick user journey

  1. Click Get Started on the home page.
  2. Choose a learning route from the class selection page.
  3. Use the smart finder to search, sort, and filter teachers.
  4. Compare two teachers and review the best recommendation.
  5. Open the map page to view teacher locations.
